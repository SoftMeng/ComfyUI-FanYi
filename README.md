# ComfyUI_FanYi

## 功能简述

一个简单的中英文翻译，使用的百度翻译.

## 使用图例

### 申请百度翻译

在百度翻译官网申请翻译API：

官网地址：https://api.fanyi.baidu.com/

![fanyi.png](image%2Ffanyi.png)

### 修改配置

在[fanyi.py](fanyi.py)中修改如下内容：

```shell
# Set your own appid/appkey.
appid = 'xxxxx'
appkey = 'xxxxxxx'
```
